== Installation ==

1. Go to `Plugins -> Add New` and search for Cowidgets - Elementor Addons
2. Activate the plugin through the 'Plugins' screen in WordPress. Make sure that you have installed the Elementor plugin.
3. Go to `Appearance -> Codeless Header/Footer/Blocks` to build a header or footer layout using Elementor.
4. After the layout is ready assign it as header or footer using the option 'Select the type of template this is'

== Frequently Asked Questions ==

= How Does This Plugin Work? =

1. Go to Appearance -> Codeless Header/Footer/Blocks to build a header or footer layout using Elementor.
2. Click on “Add New” and design a template with Elementor. 
3. Once the template is ready, set is like a header/footer/block using options “Type of Template”
	
= Can You Create a Mobile Responsive Header/Footer Using This Plugin? =

Yes, You can create the mobile responsive layout of your header using the plugin just like you do it using Elementor on other pages. Check this Elementor tutorial: [https://elementor.com/introducing-mobile-editing/](https://elementor.com/introducing-mobile-editing/)

This same applies when you are creating your Header/Footer using this plugin.
